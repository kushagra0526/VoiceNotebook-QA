# VoiceNotebook-QA

This project is a voice-based notebook application where users can record notes by speaking instead of typing. The spoken input is converted into text and stored locally in the browser.

Users can later search their notes by asking questions or entering keywords. The project also includes a document section where documents can be uploaded and searched for relevant information.

## Features
- Record voice notes
- Convert speech to text
- Store notes using local storage
- Search notes using keywords or questions
- Upload documents and search from them
- Simple and user-friendly interface

## Technologies Used
- HTML
- CSS
- TypeScript
- Vite
- Browser Speech Recognition API
- Local Storage

## How to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Install dependencies  
   `npm install`
4. Start the development server  
   `npm run dev`
5. Open the browser and use the app

## Use Case
This project can be used for quick note-taking, personal knowledge storage, and searching voice-recorded information without typing.
